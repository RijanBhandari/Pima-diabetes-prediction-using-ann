# Pima Diabetes Prediction using ANN

A deep learning project that utilizes an **Artificial Neural Network (ANN)** to predict the onset of diabetes based on diagnostic measures from the Pima Indians Diabetes dataset.

## 🛠️ Project Features
* **Neural Network Architecture:** Built using `PyTorch` with multiple fully connected (Linear) layers and ReLU activation functions.
* **Binary Classification:** Predicts whether a patient has diabetes (Outcome 0 or 1) based on 8 clinical features (Glucose, BMI, Age, etc.).
* **Data Processing:** Includes data exploration, train-test splitting, and conversion of DataFrames into PyTorch Tensors.
* **Model Persistence:** Features functionality to save the trained model as `diabeties.pt` and reload it for future inference.

## 🚀 How to Use
1. Ensure the Pima Indians Diabetes dataset is available in your environment.
2. Run the `Pima diabetes prediction using ann.ipynb` notebook to train the model.
3. The model will output accuracy metrics and save the trained weights for evaluation.
